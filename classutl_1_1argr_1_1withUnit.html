template<typename F>
class utl::argr::withUnit< F >

Allows the use of units.

If you want that a user can add k, M or G as suffix to apply a factor, you can use this argument reader. Here is a example how you could achieve it:

utl::argr::withUnit<long> myUnits{
{"", 1},
{"k", 1000},
{"M", 1000 * 1000},
{"G", 1000 * 1000 * 1000}
}
long x;
if (!args.getNextArgument(x, myUnits)) {
// handle missing argument
}
Note
The user can not eneter any number without a unit. If you want to accept pure numbers, you have to add an empty unit (""). Just like in the example. The user can always enter 0, even if there is no empty unit.
